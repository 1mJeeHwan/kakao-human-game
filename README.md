## 게임 소개

랜덤 칭호 + 직업 조합으로 캐릭터를 생성하고, 성장시켜 판매하는 게임입니다.

### 조합 예시
```
"소심한 +0 수습 백수"
"천재적인 +7 숙련 개발자"
"전설의 +15 그랜드마스터 용사"
```

## ✨ 특수 능력 시스템

### 능력 발동 순서 (강화 시)

```
1. 강화 결과 계산 (성공/실패/사망)
       ↓
2. 실패 → 성공 변환 체크
   - 🔄 실패→성공 변환: 100% 발동
   - 🍀 실패 시 30% 확률로 성공: 30% 발동
       ↓
3. 사망 → 실패 변환 체크
   - 💫 사망 1회 방지: 100% 발동
   - 🔰 사망 시 50% 확률로 방어: 50% 발동
       ↓
4. 최종 결과에 따른 처리
   - 성공: ⚡ 2레벨 상승 체크
   - 사망: 📈 레벨 유지 체크
```

### 능력 목록

| 아이콘 | 능력명 | 효과 | 발동 조건 | 확률 | 유형 |
|--------|--------|------|----------|------|------|
| 💫 | 사망 1회 방지 | 사망 → 실패 | 사망 시 | 100% | 1회용 |
| 🔰 | 사망 시 방어 | 사망 → 실패 | 사망 시 | 50% | 1회용 |
| 🔄 | 실패→성공 변환 | 실패 → 성공 | 실패 시 | 100% | 1회용 |
| 🍀 | 행운의 성공 | 실패 → 성공 | 실패 시 | 30% | 1회용 |
| ⚡ | 2레벨 상승 | +2 레벨 | 성공 시 | 100% | 1회용 |
| 📈 | 레벨 유지 | 현재 레벨 유지 | 사망 시 | 100% | 1회용 |
| 💰 | 판매가 2배 | 판매가 ×2 | 판매 시 | 100% | 1회용 |
| 💎 | 보너스 골드 | +10,000G | 판매 시 | 100% | 1회용 |
| 💸 | 비용 할인 | 강화비용 50% | 강화 시 | 100% | 1회용 |
| 🎰 | 잭팟 확률 2배 | 잭팟 확률 ×2 | 사망 시 | 100% | 1회용 |
| 🛡️ | 지원금 2배 | 파괴 지원금 ×2 | 사망 시 | 100% | 1회용 |

## 🏷️ 칭호 시스템

### 전설 등급 (2% | +100%) - 다중 능력

| 칭호 | 능력 1 | 능력 2 |
|------|--------|--------|
| 전설의 | 💫 사망 방지 | 🍀 30% 성공 |
| 신화적인 | ⚡ 2레벨 상승 | 🔰 50% 방어 |
| 불멸의 | 💫 사망 방지 | 📈 레벨 유지 |
| 태초의 | 💰 판매가 2배 | 💎 +10,000G |
| 절대적인 | 🍀 30% 성공 | 🔰 50% 방어 |
| 초월한 | 🔄 실패→성공 | ⚡ 2레벨 상승 |
| 우주의 | 🛡️ 지원금 2배 | 💫 사망 방지 |
| 심연의 | 🎰 잭팟 2배 | 💸 비용 할인 |

### 영웅 등급 (8% | +50%) - 단일 능력

| 칭호 | 능력 |
|------|------|
| 위대한 | ⚡ 2레벨 상승 |
| 고귀한 | 📈 레벨 유지 |
| 성스러운 | 🔰 50% 방어 |
| 신성한 | 💫 사망 방지 |
| 불굴의 | 🔄 실패→성공 |
| 축복받은 | 💫 사망 방지 |
| 황금빛 | 💎 +10,000G |
| 도박사 | 🎰 잭팟 2배 |

### 희귀 등급 (20% | +25%)

| 칭호 | 능력 |
|------|------|
| 행운의 | 🍀 30% 성공 |
| 알뜰한 | 💸 비용 할인 |
| 기타 10개 | 능력 없음 |

### 고급/일반 등급

- 고급 (30%): +10% 보너스, 능력 없음
- 일반 (40%): 보너스 없음, 능력 없음

## 📊 판매가 계산 로직

### 공식

```javascript
판매가 = 기본가 × (1 + 칭호보너스 + 직업보너스)
```

### 기본가 계산

| 레벨 구간 | 계산 방식 |
|-----------|----------|
| 1~6강 | `2^level × 10` |
| 7~11강 | 6강 기준 × `1.5^n` × `2^(level-6)` |
| 12~20강 | 11강 기준 × `1.8^n` × `2^(level-6)` |
