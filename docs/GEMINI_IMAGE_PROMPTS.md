## 생성 팁

1. **일관성 유지**: 모든 이미지에 동일한 기본 프롬프트 사용
2. **배경**: 투명 또는 흰색 단색 배경 권장
3. **크기**: 256x256 또는 512x512 (나중에 리사이즈 가능)
4. **저장**: PNG 형식 (투명 배경 지원)
5. **네이밍**: 영문 소문자 + 언더스코어 (예: office_worker.png)

## 이미지 호스팅 옵션

### 1. ImgBB (무료)
- 무료 32MB 업로드
- API 제공
- 영구 호스팅

### 2. Cloudinary (무료 티어)
- 25GB 스토리지
- CDN 제공
- 이미지 변환 API

### 3. GitHub Raw
- 레포에 직접 저장
- 무료
- `https://raw.githubusercontent.com/user/repo/main/images/`

## 이미지 업로드 후 설정

`utils/jobImages.js` 파일에서 `IMAGE_BASE_URL` 수정:

```javascript
// ImgBB 사용 시
const IMAGE_BASE_URL = 'https://i.ibb.co/your-album/';

// Cloudinary 사용 시
const IMAGE_BASE_URL = 'https://res.cloudinary.com/your-cloud/image/upload/jobs/';

// GitHub Raw 사용 시
const IMAGE_BASE_URL = 'https://raw.githubusercontent.com/username/kakao_game_1/main/public/images/jobs/';
```

또는 환경변수로 설정:
```bash
export JOB_IMAGE_BASE_URL="https://i.ibb.co/your-album/"
```
